# Noor Brand V2

Premium bilingual academic portfolio for Noor Muhannad Ahmed.

## Main features
- Custom Noor Muhannad brand logo
- English / Arabic
- Light / dark mode
- Intro logo animation
- Animated cyber-network background
- Research metrics and animated counters
- Research, Research Lab, Timeline, Credentials, Contact
- IEEE membership card
- iAIMS 2026 prepared paper PDF
- QR code to live portfolio
- Responsive design

## Deployment
Upload all files and folders to:
https://github.com/noor-muhannad/noor-academic-portfolio

Keep the `assets` folder structure intact.

## Important
Add the latest CV PDF to the project root as:
`cv.pdf`
